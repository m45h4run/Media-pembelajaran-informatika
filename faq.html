Cara Penggunaan

1. Tombol tentang pada gambar diatas, digunakan untuk melihat tentang sekolah singkat

2. Tombol WhatsApp digunakan untuk menghubungi Guru pengampu.

3. Icon pdf pada gambar di atas digunakan untuk mendownload file kompetensi dasar

4. Icon Rumah digunakan untuk kembali ke tampilan awal/beranda

5. Icon Buku digunakan untuk melihat materi-materi yang ada

6. Icon Kertas/note digunakan untuk mengerjakan kuis, sebelum mengerjakan kuis anda akan diminta untuk memasukkan Nama dan NIS

7. Icon Tanya/tanda tanya digunakan untuk menampilkan halaman ini/ halaman FAQ

8. Setelah anda menekan tombol 'buku' pada nomor (5) maka akan ditampilkan gambar diatas, yang mana terdapat card gambar materi, anda dapat menekan gambar untuk menampilkan judul dari setiap materi yang ada. Untuk menilhat isi materi anda dapat menekan icon (rantai).

9. Setelah anda menekan icon 'rantai' pada nomor (8) maka akan di tampilkan materi sesuai yang anda pilih.

10. Pada bagian atas materi terdapat tanda '<-' dan tertulis 'Materi' itu digunakan untuk kembali pada halaman awal materi pada nomor (8)

11. Pada bagian bawah materi 'paling bawah' terdapat tanda '<-' dan tanda '->' itu digunakan untuk melihat materi sebelumnya dan materi selanjutnya.

12. Setelah anda menekan tombok Kertas/note pada nomor (6) anda akan diminta untuk memasukkan nama dan NIS untuk membuka kuis, silahkan di isi dan tekan tombol 'oke'

Ikuti pentujuk kuis untuk mengerjakan. Sampai muncul tampilan pada gambar di bawah ini.
